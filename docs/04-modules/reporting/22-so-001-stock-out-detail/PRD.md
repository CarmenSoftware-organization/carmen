# Stock Out Detail Report - Product Requirement Document

**Report ID:** SO-001
**Report Name:** Stock Out Detail Report
**Module:** Stock Out (SO)
**Priority:** Medium
**Status:** Specification from MSD

## Executive Summary

Display detailed information of all stock issuances including various transaction types (Issue, Transfer Out, Adjustment, Consumption).

## Key Features

### Transaction Types
- Issue (to departments/operations)
- Transfer Out (to other locations)
- Adjustment (inventory adjustments)
- Consumption (manufacturing usage)

### Filter Parameters
- Date Range: Stock out date range
- Location: Source location filter
- Transaction Type: Type of stock out movement
