# Inventory Balance Report - Product Requirement Document

**Report ID:** INV-001
**Report Name:** Inventory Balance Report
**Module:** Inventory Management
**Priority:** Critical
**Status:** Specification from MSD

## Executive Summary

Display current inventory balances by location and product with real-time or as-of-date snapshots.

## Key Features

- On Hand Quantity
- Reserved Quantity
- Available Quantity
- Unit Cost and Total Value
- Reorder Level Status
- Multi-location support
