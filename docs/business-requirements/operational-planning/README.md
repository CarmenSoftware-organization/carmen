## Module Overview

The Operational Planning module manages recipes, ingredients, and forecasting. Key features include:

- Recipe creation and management
- Ingredient specifications
- Recipe costing and cost analysis
- Recipe versioning
- Yield management
- Menu planning support
- Cost forecasting

## Key Stakeholders

- Chefs and Kitchen Managers
- Menu Planners
- Cost Controllers
- Finance Team
- Procurement Team
