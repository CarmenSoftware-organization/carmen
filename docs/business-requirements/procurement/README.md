## Module Overview

The Procurement module manages the complete procurement lifecycle from purchase requests to goods receipt and credit notes. Key features include:

- Purchase Request creation and approval workflows
- Purchase Order generation and management
- Goods Received Note (GRN) processing
- Credit Note handling
- Vendor price list management
- Price comparison tools
- Budget control and validation
