## Module Overview

The Inventory Management module maintains accurate inventory records across all locations. Key features include:

- Stock In/Out management
- Physical count and spot check management
- Period-end processing
- Inventory adjustments
- Stock replenishment
- Multi-location inventory tracking
- Lot tracking system
- Inventory valuation
