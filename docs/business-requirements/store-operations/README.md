## Module Overview

The Store Operations module manages day-to-day store activities and inventory usage. Key features include:

- Store requisitions
- Issue management
- Wastage tracking and reporting
- Usage monitoring
- Transfer between locations
- Return processing

## Key Stakeholders

- Store Managers
- Department Heads
- Kitchen Managers
- Inventory Controllers
- Finance Team
