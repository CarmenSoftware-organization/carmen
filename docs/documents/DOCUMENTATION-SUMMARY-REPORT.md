## üèóÔ∏è Technical Architecture Summary

> **üìê For comprehensive technical architecture with diagrams and code examples, see [System Architecture](./architecture/SYSTEM-ARCHITECTURE.md)**

### Architecture Highlights

**5-Layer Architecture Pattern:**
1. Presentation Layer - React Components, Responsive UI
2. Application Layer - Next.js App Router, Server Components
3. Business Logic Layer - Server Actions, Workflow Engine
4. Data Access Layer - Type System, Mock Data, Validators
5. Cross-Cutting Concerns - Auth, RBAC, Logging, Errors

**Technology Stack:**
- Next.js 14 + React 18 + TypeScript 5.8+
- Tailwind CSS + Shadcn/ui + Radix UI
- Zustand (Global State) + React Query (Server State)
- React Hook Form + Zod Validation

**Key Architectural Decisions:**
- ‚úÖ Server-First Approach (Server Components by default)
- ‚úÖ Centralized Type System (`/lib/types/`) with guards and validators
- ‚úÖ Centralized Mock Data (`/lib/mock-data/`) with factory functions
- ‚úÖ Role-Based Access Control with 7 user roles
